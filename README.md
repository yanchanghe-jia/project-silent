\# PROJECT SILENT — 沉默之眼



一款基于浏览器的复古心理恐怖解谜游戏，模拟 Windows 98 操作系统，结合 ARG 元素，玩家需要在表里两个世界中寻找线索，揭开一桩神秘失踪案的真相。



\---



\## 🎮 游戏简介



你扮演一名普通的年轻人陈垣，因为朋友林越的离奇失踪，无意间接触到了一个神秘的压缩包。运行之后，你的电脑从现代的 Windows 11 坠入了一个诡异的 1998 年警局档案系统。在这个被“沉默之眼”注视的里世界中，你需要翻阅大量真假难辨的案卷、物证、通讯记录，拼凑线索，同时警惕那个正在窥视你的未知存在。



\---

\## 📜 故事背景



1998 年 12 月 24 日，中国北方小镇“静湖镇”的天文观测站发生了一起研究员失踪案。警方草草结案，但真相被埋藏在尘封的档案中。二十多年后，一个名为“谜域”的论坛上出现了当年的内部文件，而你的朋友林越在前往调查后音讯全无。你在他遗留的电脑里发现了一个压缩包，运行后，整个系统回到了 1998 年……



\---



\## 🧩 核心玩法



\- \*\*双时空切换\*\*：表世界（Windows 11 风格）与里世界（Windows 98 风格）交替进行。

\- \*\*多应用检索\*\*：通过案宗索引、人员档案、物证追踪、通讯记录、坐标定位等应用查询线索。

\- \*\*线索拼接\*\*：关键信息被分散在日记、简报、内部备忘录、值班日志等大量文本中，需要玩家自行组合。

\- \*\*多结局系统\*\*：根据玩家的操作和选择，触发普通、真结局、坏结局等不同结局。

\- \*\*恐怖氛围\*\*：随机日期崩坏、屏幕震颤、隐藏弹窗、反窥探遮罩等元素营造持续的不安感。

\## 🛠️ 技术栈



\- 纯前端：HTML、CSS、JavaScript

\- 无需任何框架或构建工具，开箱即用

\- 使用 localStorage 进行状态持久化

\- 通过 postMessage 实现 iframe 跨页面通信

\- 支持全屏模式与音频占位
